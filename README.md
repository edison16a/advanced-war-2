````markdown
# Advanced War 2 (HTML Remake) — 2 Player Mode

A lightweight, browser-based remake inspired by classic turn-based strategy gameplay. Built in **plain HTML/CSS/JavaScript** for quick local play with **2 players on the same device** (hotseat).

## Features
- **2-player hotseat** turns (Player 1 vs Player 2)
- Runs locally — **no install, no server**
- Simple controls and fast restart loop
- Clean, minimal UI for quick gameplay
````

Then open `index.html`.

## How to Play (2 Player Hotseat)

* Players take turns on the same computer.
* The UI will indicate whose turn it is.
* Perform your actions, then click **End Turn** to pass control.

## Controls

*(Adjust these bullets if your project has specific bindings.)*

* **Click** to select units / tiles
* **Click** again to move / confirm actions
* **End Turn** button to switch players
* **Restart** button to reset the match
